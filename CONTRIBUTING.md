# Contributing to PeachBot Core

## Overview

PeachBot Core is an evolving system focused on distributed, edge-first intelligence architectures.

Contributions are welcome from individuals working in:

- Edge AI systems  
- Distributed learning (federated systems)  
- Graph Neural Networks (Edge-GNN)  
- Embedded and edge hardware systems  
- Biological and adaptive computation models  

---

## Contribution Scope

At this stage, contributions are most relevant in:

- Core architecture modules (FILA, SBC, orchestration)  
- Edge system optimization  
- Model efficiency for constrained environments  
- Deployment and infrastructure tooling  

---

## How to Contribute

1. Open an issue describing:
   - The problem or idea  
   - Proposed approach  

2. Wait for discussion and alignment  

3. Submit a pull request with:
   - Clear description  
   - Minimal and focused changes  
   - Documentation updates where applicable  

---

## Guidelines

- Maintain consistency with system architecture (see `docs/architecture.md`)  
- Avoid introducing unnecessary dependencies  
- Focus on deployment-oriented solutions  
- Keep implementations modular and extensible  

---

## Code & Documentation

- Code should be readable and modular  
- Documentation updates should accompany architectural changes  
- Use clear and consistent terminology (FILA, SBC, Edge-SoC, etc.)  

---

## Collaboration

PeachBot Core is being developed as a **system-level platform**, and contributions are expected to align with:

- Distributed intelligence principles  
- Edge-first execution  
- Real-world deployment constraints  

---

## Note

This repository is under active development.  
Contribution processes may evolve as the system matures.
