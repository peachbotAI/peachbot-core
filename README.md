# PeachBot Core

## Overview

PeachBot Core is the foundational system layer of the PeachBot platform, designed as a distributed, edge-first intelligence architecture.

It integrates:
- Federated Intelligence & Learning Architecture (FILA)
- Synthetic Biological Computation (SBC)
- Edge System-on-Chip (SoC) integration
- Platform orchestration and governance

---

## Repository Structure

- `docs/` → System documentation and research foundations
- `core/` → Core system frameworks (FILA, SBC, orchestration)
- `interfaces/` → Domain-specific integration layers
- `models/` → AI and signal processing models
- `deployment/` → Deployment configurations and infrastructure
